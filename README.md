# Visualizing Folklore Classics in the Age of Digital Intelligence: A Case Study of Wu Bing’an’s Collected Works

Repository Status & Limitations
This repository contains the core source code for the web application associated with our research. Please note the following constraints regarding this codebase:

Partial Upload & Execution: Certain backend logic modules, databases, and proprietary algorithms are not included or executable in this repository due to security, privacy, and dependency constraints.

Copyrighted Materials: PDF documents and specific external resources have been intentionally redacted from this repository to comply with copyright laws.

Live Deployment: A live, public version of this website is not currently available. Website deployment requires a mandatory government ICP (Internet Content Provider) filing in China, which is a pending administrative process.

The code provided here serves to demonstrate the core structural implementation and validate the methodology proposed in our study.
